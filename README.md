# Arduino LCD 16x2 Clock

นาฬิกาดิจิทัลสำหรับ Arduino UNO และ LCD Keypad Shield 16x2

## Features

- แสดงวัน/เดือน/ปี
- แสดงชั่วโมง/นาที/วินาที
- ตั้งค่าวันที่และเวลาได้จากปุ่มบน LCD Keypad Shield
- ระบบ Leap Year
- กระพริบช่องที่กำลังแก้ไข

## Hardware

- Arduino UNO
- LCD Keypad Shield 16x2

## Pin Configuration

LCD:
RS = 8
EN = 9
D4 = 4
D5 = 5
D6 = 6
D7 = 7

Buttons:
A0 (Analog Input)

## Controls

RIGHT : เข้า/ออกโหมดแก้ไข

UP : เพิ่มค่า

DOWN : ลดค่า

LEFT : เลื่อนช่องย้อนกลับ

SELECT : เลื่อนไปช่องถัดไป
